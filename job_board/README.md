# job_board

## To set this project up:
```
git clone git@github.com:LaurenQurashi/JobBoard.git
npm install
```

### To run the app on your local host:
```
npm run serve
```

### To run the tests:
```
npm run test
```

### To run the Linters and fix files:
```
npm run lint
```

### To run the E2E (Cypress) tests:
```
npm run test:e2e
```

### To run the Unit (Mocha) tests:
```
npm run test:unit
```
